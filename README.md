# Restaurant Ordering System

A Java desktop application simulating a restaurant ordering system, 
built with Swing GUI and ArrayList for data management.

## Features
- Browse menu items by category
- Add items to cart and manage orders
- Calculate total price automatically
- Simple and clean GUI interface

## Tech Stack
- Java
- Java Swing (GUI)
- ArrayList (data management)
- OOP design principles

## How to Run
1. Clone the repository
   git clone https://github.com/jiga5633/Resaurant.git
2. Open in IntelliJ IDEA or Eclipse
3. Run the main class
